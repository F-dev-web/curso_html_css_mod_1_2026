Principais Formatações em HTML

Negrito/Destaque

Este exemplo mostraremos uma palavra em negrito de forma não semântica através da tag < b > < /b >

Neste outro exemplo uma palavra em destaque através da tag < strong > < /strong > com semântica.

Itálico/Ênfase

Agora usaremos como exemplo uma palavra em itálico mais uma vez sem semântica utilizando a tag < i > < /i >

Para obtermos e mesmo resultado com o itálico de com semântica devemos usar a tag < em > < /em >

Texto Marcado

Podemos também criar um texto marcado usando a tag < mark > < /mark >

Texto grande e Texto pequeno

Estamos criando um parágrafo com texto grande com a tag < big > < /big >, já obsoleta e um texto pequeno com a tag < small > < /small > ainda em uso no HTML

Texto Deletado

Podemos marcar um texto como um texto excluído, com a tag < del > < /del >, indicando que ele deve ser lido, mas não considerado

Texto Inserido

Aqui marcaremos um texto como inserido, com a tag < ins > < /ins >, para dar ênfase e indicar que foi adicionadom à posteriore

Diferentemente desse exemplo onde o texto se destaca, através da tag < u > < /u >, apenas pela forma e não pela semântica

Texto Sobreescrito

Usado para darmos sentido de expoentes ex:
2x =22 fica 2x = 22, usando a tag < sup > < /sup >

Texto Subescrito

Neste exemplo para dea significado de índice de fórmulas científicas por ex:
H2O fica H2O, por causa da tag < sub > < /sub >
