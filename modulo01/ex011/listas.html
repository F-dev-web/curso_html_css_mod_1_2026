Listas Ordenadas

São listas criadas com uma ordem ou sequência a ser seguida, e são criadas com a tag < ol > < /ol >, que indicará uma lista ordenada e dentro desta tag serão inclusos itens que por sua vez ficarão entre a tag < li > < /li >.
Sendo uma tag "ol" para a lista toda e uma tag "li" para cada item da lista. A parte final da tag (o seu fechamento) < /li > a partir do HTML5, passou a ser opcional, não sendo necessário sua inclusão.

Exemplos de listas ordenadas:

  1. Acordar
  2. Escovar os dentes
  3. Tomar café
  4. Ir trabalhar

Podemos também adicionar parmetros para a ista ordenada dentro da tag "ol"
< ol type= A > < /ol > alterando os índices da lista de números para letras

  1. Acordar
  2. Escovar os dentes
  3. Tomar café
  4. Ir trabalhar

Se o parâmetro for o numeral "1" a lista fica com os índices originais, se for a letra "A" maiúscula como o exemplo os índices serão letras maiúsculas, se forem letras minúsculas eles mudarão para letras minúsculas também
os parâmetros "I" e "i" respectivamente se referem a numerais romanos

  1. Acordar
  2. Escovar os dentes
  3. Tomar café
  4. Ir trabalhar
  1. Acordar
  2. Escovar os dentes
  3. Tomar café
  4. Ir trabalhar

Listas não ordenadas

Como o próprio nome já diz são listas que não precisa seguir uma sequência ou ordem como por exemplo lista de compras
e são criadas com a tag < ul > < /ul >, para a tipificação da lista e a tag < li > < /li > para os itens da lista.

  • pão
  • leite
  • açúcar
  • café
  • margarina

Como na listas ordenadas nas não ordenadas também podemos alterar o formato dos índices acrescentando no parâmetro "type" uma das opções: disc, circle ou square, sendo disc o formato padrão sendo desnecessário informá-lo se for a opção desejada

  • pão
  • leite
  • açúcar
  • café
  • margarina
  • pão
  • leite
  • açúcar
  • café
  • margarina

Essas listas podem ser combinadas em ordenadas e não ordenadas ou ordenadas com não ordenadas
