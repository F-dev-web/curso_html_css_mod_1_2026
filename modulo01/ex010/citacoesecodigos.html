O comando document.getElemrntById('teste') pertence à linguagem JavaScript, escrito dessa forma cada uma das letras utilizadas ocupam espaços de tamanhos diferentes, resolvemos essa situação dessa forma:
colocando entre a tag < code > < /code > document.getElementById('teste'). Ficando < code > document.getElementById('teste') < /code > para obtermos o resultado abaixo:
document.getElementById('teste')

Mas a tag < code > < /code > não cria identação usadas em outras linguagens como no python por exemplo para obtermos uma exibição em modelo de identação precisamos colocarmos todo o comando inclusive a tag < code > < /code > dentro de uma outra tag a < pre > < /pre >
